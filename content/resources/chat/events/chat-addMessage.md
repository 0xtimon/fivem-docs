title: chat:addMessage

## About
Triggering this event allows you to send a chat message to this client.
Message object structure:

```lua
message = {
  color = color,
  multiline = true,
  args = {author, otherArgs...}
}
```

## Name
```
chat:addMessage
```

Parameters
----------

```
object message
```
